# CERICar – Projet Web L3

CERICar est une application web de covoiturage développée en Yii2 dans le cadre de la Licence 3 Informatique.  
Le projet utilise la base de données `fredouil` fournie par l’enseignante et s'appuie sur une modélisation UML complète (cas d’utilisation, diagramme de classes, séquences).

## Contenu du projet
- Code Yii2 (MVC)
- Modèles liés à la base de données PostgreSQL
- Diagrammes UML
- Documentation du projet
